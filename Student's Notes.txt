disini saya membuat sebuah function untuk handling ketika navbar bergulir dan memasuki,id yang di tuju maka akan memiliki background debgab cara memanipulasi background dari solid ke transparent,
cara mengakses nya pertama saya mendeklarasikan id element html navbar lalu saya mendeklarasikan juga id content yaitu tempat dimana ketika id navbar akan memiliki background solid atau berwarna dengan background awal transparent lalu 
function handle scroll sebagai setup atau logika memanipulasi navbar yang akan di jalankan kondisi yang saya gunakan adalah if (window.scrollY > content.offsetTop) yaitu ketika jumlah pixel atau jarak yang sudah di scroll melebihi atau sama dengan jarak content maka kondisi akan terpenuhi dan background akan menjadi solid
lalu jika tidak terpenuhi maka background akan transparent kembali, selain itu saya menerapkan atau membuat variable baru yaitu initialNavbar ini berfungsi agar ketika web si refresh dan posisi navbar berada di bawah maka tidak akan transparent seperti di awal mula web di render 
dan yang terakhir agar navbar dapat berubah sesuai setup yang sudah di buat sebelumnya saya memanggil funsi window.addEventListener("scroll", handleScroll); agar web membaca scroll dan menjalan kan funsi yang telah di buat 
lalu untuk function yang kedua saya membuat funsi untuk button untuk menampilkan /mengganti materi yang di inginkan langkah pertama saya mendeklarasikan id kedua button dan div atau elemen yang akan di manipulasi lalu saya membuat variable awal selection yang bertipe boolean untuk menentukan conten yang akan di tampilkan lalu sya membuat function klik yang dimana jika button di klik akan mengubah variable selection jika selection tidak sama dengan nilai button yang di inginkan jika nilai selection sudah sesuai dengan yang di inginkan maka function akan langsung mengubah elemen tanpa mengganti nilai selection walaupun pada dasarnya hal ini tidak akan terlihat jika anda mengklik tomboll lebih dari sekali
